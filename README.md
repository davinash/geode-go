# geode-go
geode-go provides Go package for connecting with [Apache Geode](https://geode.apache.org/).

## Install
```bash
go get -u github.com/davinash/geode-go
```

## Region APIs Supported
* Put
* Get
* PutAll
* GetAll
* PutIfAbsent
* Remove
* Size
* Clear
